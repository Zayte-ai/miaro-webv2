# 3D Models Directory

This directory will contain the 3D models for the clothing items.

## Expected files:
- white-tee.glb
- hoodie.glb
- jeans.glb
- jacket.glb

## Model specifications:
- Format: GLB (recommended) or GLTF
- Optimization: Models should be optimized for web display
- Size: Keep file sizes reasonable for web loading (< 5MB per model)
- Textures: Include embedded textures when possible
- Scale: Models should be appropriately scaled

## Tools for creating/converting models:
- Blender (free, open-source)
- Three.js Editor
- Online GLB converters

Jakob can provide these 3D models when ready, and they should be placed in this directory with the exact filenames listed above.
